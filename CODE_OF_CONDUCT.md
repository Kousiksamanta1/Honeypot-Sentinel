# Code of Conduct

## Our Standard

This project welcomes respectful, constructive participation from people of
all backgrounds and experience levels.

Positive participation includes:

- discussing ideas and technical tradeoffs with empathy
- giving actionable feedback and accepting correction gracefully
- respecting privacy, authorization boundaries, and responsible disclosure
- keeping security examples legal, sanitized, and relevant to the project

Unacceptable behavior includes harassment, discrimination, personal attacks,
publishing private information, deliberately unsafe security guidance, or
using project spaces to facilitate unauthorized access.

## Enforcement

Project maintainers may edit, hide, or remove contributions that violate this
standard and may temporarily or permanently restrict participation.

Report conduct concerns privately to `kousik83@gmail.com`. Include links and
context, but do not forward unnecessary personal or sensitive information.

Maintainers will review reports fairly, protect reporter privacy where
possible, and take action proportionate to the behavior.
